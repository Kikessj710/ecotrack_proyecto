# 🌿 EcoTrack

## Plataforma de Conciencia Ambiental

---

### 📌 ¿Qué es EcoTrack?

**EcoTrack** es una plataforma web diseñada para ayudar a las personas a conocer y reducir su impacto ambiental. A través de una calculadora de huella de carbono, consejos prácticos y contenido educativo, buscamos convertir la conciencia ecológica en acción.

> *"Medir para entender, entender para mejorar."*

---

### 🎯 Objetivo del Proyecto

Desarrollar una herramienta accesible, visual y fácil de usar que permita a cualquier persona:

- **Calcular** su huella de carbono personal
- **Identificar** qué hábitos generan más emisiones
- **Aprender** cómo reducir su impacto con cambios simples
